# Ansible Java Role

Manages installation of Java JREs and JDKs. It supports both OpenJDK and Oracle JRE and JDK versions:
- 6
- 7
- 8
- 9

# Role variables

## Configure default Java implementation.
Choices:
- oraclejdk_version
- openjdk_version

#### openjdk_version
    - 6
    - 7
    - 8
    - 9

#### oraclejdk_version
    - 6
    - 7
    - 8
    - 9
